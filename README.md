
# Keyboard teacher

A simple C++/Qt project for improoving your typing skills. Generates text to type automatically, shows the statistics(speed, accuracy, time), shows the correct key areas for each finger, provides many languages to train.



## Features

- Supported languages: german, french, arabic, belarussian, ivrit, russian, italic, english
- Internet connection independency
- Fullscreen mode
- Colored keyboard with different area for each finger area
- Incorrect input handling
- Statistics
- Automatic text generation



## Technologies

- C++: algorythms
- Qt: interface
